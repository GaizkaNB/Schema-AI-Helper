# Schema.org Documentation AI Knowledge Pack

This folder contains generated Markdown knowledge packs built from Schema.org documentation pages. The files are intended to be uploaded directly into an AI assistant or custom GPT knowledge base so builders can reuse Schema.org reference material without first crawling and cleaning the site themselves.

This repository is not an official Schema.org project. The source documentation comes from [Schema.org](https://schema.org/), and reuse of the underlying Schema.org schema content is governed by the [Schema.org Terms of Service](https://schema.org/docs/terms.html), which reference the Creative Commons Attribution-ShareAlike 3.0 license for the schema.
